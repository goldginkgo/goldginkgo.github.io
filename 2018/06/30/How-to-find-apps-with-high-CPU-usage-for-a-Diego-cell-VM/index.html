How to find apps with high CPU usage for a Diego cell VM.

Posted on 2018-06-30 | In Cloud
| 5.7k | 5 mins.

According to Understanding Container Security, each container in Cloud Foundry use a fair share of CPU relative to the other containers. However, an app container can use much higher CPU resources when other containers in the same Diego Cell VM are idle if we don’t set limitation for CPU usage of app containsers.

It would be rather difficult to figure out the apps that is causing the trouble if we don’t have application level monitoring.

One way to achieve our goal is to use Cloud Foundry top plugin. But here I also want to show the procedures on how to do it manually.
